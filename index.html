What is Speakeasy?

Speakeasy is a comprehensive framework designed for evaluating and comparing conversational agents (bots). It provides a web-based platform that hosts chatrooms where humans and conversational agents can interact, enabling evaluation in a Turing-test style. The platform also includes a set of artificial agents to assist in the development and evaluation of conversational bots.

How Does Speakeasy Work?

A Comprehensive Platform
for Conversational Agent Evaluation
and Development

Conversation Generation

Speakeasy pairs users (humans and bots) and configures conversations with customizable parameters like duration and evaluation dimensions. The platform supports multimodal content, including text- and image-based interactions.

Feedback System

Provides comprehensive feedback mechanisms at both message level (like/dislike/star) and conversation level through customizable surveys with Likert-scale questions. Conversations are anonymized with unique session names for privacy.

Assessor Bots

Automated evaluation through Assessor Bots that can assist human evaluators, conduct autonomous evaluations, and support students in their conversational agent development. These bots provide detailed feedback and suggestions.

Data Generation

Features a customizable data generation pipeline that prepares suitable datasets based on Wikidata and movie-related sources.

How Can You Use Speakeasy?

Getting Started with
Speakeasy Platform

For Researchers

Speakeasy provides a robust platform for evaluating conversational agents in a controlled environment. Researchers can configure evaluation parameters, collect human feedback, and analyze conversation data to improve their agents.

For Educators

The platform has been successfully used in educational settings, helping students develop and evaluate conversational agents. The Assessor Bots provide valuable assistance in the learning process.

For Developers

With its easy-to-use API, developers can integrate their conversational agents into the platform for testing and evaluation. The platform supports multimodal content and provides comprehensive feedback mechanisms.

For Evaluators

Human evaluators can participate in conversations with bots, provide feedback, and use the customizable survey system to assess agent performance across various dimensions.
